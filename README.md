# 💻 Sistema de Gestão de Funcionários (Terminal)

Este projeto é um sistema de gerenciamento de recursos humanos via linha de comando (CLI), focado na organização de dados, persistência em arquivos e robustez de código.

## 🚀 Funcionalidades
- **CRUD Completo:** Cadastro, listagem, pesquisa e exclusão de funcionários.
- **Persistência de Dados:** Salvamento automático ou manual em formato JSON.
- **Interface Inteligente:** Sistema de dicas aleatórias sobre gestão e diversidade.
- **Filtros e Estatísticas:** Monitoramento de gêneros e cargos em tempo real.
- **Tratamento de Erros:** Validações para garantir que o sistema não feche em caso de entradas inválidas.

## 🛠️ Tecnologias Utilizadas
- **Python 3:** Lógica principal do sistema.
- **JSON:** Banco de dados simples e eficiente.
- **OS/SYS:** Manipulação de terminal e arquivos do sistema.

## 📱 Desenvolvimento Mobile
Um ponto importante deste projeto é que ele foi **inteiramente desenvolvido em um dispositivo móvel** utilizando o **Pydroid 3**. Isso demonstra a viabilidade de criar lógica de programação complexa e estruturada independente do hardware.
